Functional CSS is:



  • Pure (no side effects)
  • Composable
  • Immutable
  • Transparent (rem vs em)
Pure - class for background color sets the background color only
Pure - in standard CSS one class changes 10 properties and 5 in children due to cascading
Composable - small, reusable blocks
Immutable - property once declared, won't change (no specificity fight)
Transparent - you know what a class does, no matter where on page
